# 🎯 ML Prediction API with FastAPI

This project is a Machine Learning API built using FastAPI.  
The API takes user input and returns predictions from a trained ML model.

---

## 🚀 Features

- Machine Learning model prediction
- REST API using FastAPI
- Input validation using Pydantic
- Easy deployment
- JSON response

---

## 🛠️ Technologies Used

- Python
- FastAPI
- Scikit-learn
- Pandas
- NumPy
- Uvicorn
